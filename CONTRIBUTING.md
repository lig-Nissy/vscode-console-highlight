# Contributing

プロジェクトへの貢献を歓迎します！

## 開発環境のセットアップ

```bash
git clone https://github.com/YOUR_USERNAME/vscode-console-highlight.git
cd vscode-console-highlight
npm install
```

## 開発の流れ

1. このリポジトリをフォーク
2. フィーチャーブランチを作成 (`git checkout -b feature/amazing-feature`)
3. 変更をコミット (`git commit -m 'Add some amazing feature'`)
4. ブランチをプッシュ (`git push origin feature/amazing-feature`)
5. Pull Requestを作成

## デバッグ

1. VSCodeでプロジェクトを開く
2. `F5`を押してデバッグモードで起動
3. 新しいVSCodeウィンドウが開き、拡張機能がロードされる

## コードスタイル

- TypeScriptを使用
- `npm run compile`でエラーが出ないこと

## バグ報告・機能リクエスト

GitHubのIssuesで報告してください。
